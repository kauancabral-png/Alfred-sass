# 🎩 Lista de Variáveis para Railway (Copie e Cole)

Mestre, entre na sua conta da **[Railway](https://railway.app/dashboard)**, clique no projeto do seu **Backend** (NodeJS/Express), vá na aba **"Variables"** e clique em **"New Variable"** para cada uma destas:

| NOME DA VARIÁVEL (Name) | VALOR (Value) | O que é? |
| :--- | :--- | :--- |
| **`SMTP_HOST`** | `smtp.gmail.com` | Servidor de E-mail (Se você usa Gmail) |
| **`SMTP_PORT`** | `465` | Porta de Segurança (SSL) |
| **`SMTP_USER`** | `seu-email-aqui@gmail.com` | **O seu e-mail de envio** |
| **`SMTP_PASS`** | `xxxx yyyy zzzz wwww` | **O código de 16 letras que você pegou no Google** |
| **`JWT_SECRET`** | `ALFRED_SAAS_SECRET_2026` | Chave de Segurança das Contas |
| **`KIRVANO_WEBHOOK_TOKEN`** | `ALFRED_MESTRE_SAAS_2026` | Token de Segurança da Kirvano |

---

### ⚠️ Dica de Ouro:
Após colar tudo, o Alfred vai fazer um pequeno **"reboot"** na Railway sozinho para aplicar os novos poderes.

👉 **O senhor já pegou aquele código de 16 letras do Google?** (Sem ele, o Alfred não consegue abrir as portas do e-mail!) 🎩🥂🍷🍾🕵️‍♂️
